Juego de Adivinar el Número en Java

Descripción

Este programa genera un número aleatorio entre 1 y 100.
El jugador debe adivinar el número ingresando intentos desde el teclado.

El sistema indica si el número secreto es mayor o menor hasta que el jugador lo adivine.

Cómo funciona

1. El programa genera un número secreto.
2. El usuario ingresa un número.
3. El sistema da pistas (mayor o menor).
4. El juego termina cuando el jugador acierta.

Tecnologías utilizadas

* Java
* Scanner
* Random
